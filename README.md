## 📖 Overview
This project is a premium personal portfolio website designed to showcase my skills, projects, education, and achievements as a Full Stack Developer. It features a modern dark theme with elegant glassmorphism effects, dynamic CSS keyframe animations, and custom JavaScript interactivity (including an automated typing effect and 3D tilt hover effects).

## 🚀 Live Demo
Experience the live version of the portfolio here:
👉 **[Live Demo](https://varung-coder.github.io/PRODIGY_WD_04/)**

## ✨ Key Features
- **Premium Glassmorphism Aesthetics**: Modern UI with frosted glass cards, subtle borders, and smooth backdrop blurring.
- **Dynamic Typing Animation**: Engaging hero section featuring a custom typing loop highlighting various developer roles.
- **Custom Interactive Particles**: A lightweight, vanilla JavaScript-based background particle system.
- **3D Tilt Hover Effects**: Project cards react to mouse movements, creating a premium physical tilt and glow effect.
- **Scroll Reveal Animations**: Smooth fade-in and slide-up animations triggered on scroll, powered by the Intersection Observer API.
- **Fully Responsive Design**: Perfectly adapted layouts for mobile, tablet, and desktop viewports using CSS Grid and Flexbox.
- **Sticky Navigation**: A responsive navigation bar that adapts to scrolling with an integrated mobile hamburger menu.

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3, Vanilla JavaScript (ES6+)
- **Design System**: Custom CSS Variables, Glassmorphism, CSS Grid/Flexbox
- **Icons**: FontAwesome
- **Deployment**: Vercel & GitHub Pages
